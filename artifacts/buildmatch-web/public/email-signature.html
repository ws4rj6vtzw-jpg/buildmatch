How to add this to Gmail

  1. Click Copy Signature below.
  2. Open Gmail → Settings (gear icon) → See all settings → Signature.
  3. Create a new signature and paste (Ctrl+V / Cmd+V) into the editor box.
  4. Save changes. Done.

For Outlook: File → Options → Mail → Signatures, then paste into the signature editor.
